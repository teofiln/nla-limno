# nla-limno

NLA dataset for limnology course at University of Costa Rica

All data are downloaded from: https://www.epa.gov/national-aquatic-resource-surveys/data-national-aquatic-resource-surveys.

The repo includes some R and QMD scripts for teaching data manipulation in limnological context.

## Como descargar este repositorio

### `git`

Con git es facil:

```
# Via HTTPS:
git clone https://github.com/teofiln/nla-limno.git
```

### No `git`

Tambien facil:

1. Navegar a https://github.com/teofiln/nla-limno
2. Haga clic al boton `Code`
3. Haga click al boton `Download ZIP`
